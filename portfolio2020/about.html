-How I learned to code-

After a selection procedure with 160 applications and 20 spots available I was selected to join the free Restart Network web development program in autumn 2018. The program was about 60 hours studying per week for six months. The Restart students were mainly expats and refugees bringing all their diffirent cultures to this pressure cooker coding campus which made it even more interesting. Because besides code I learned so much about other cultures. We had stand-up meetings and algorithm classes every day. We had masterclasses and soft skill classes every week, plus every two weeks we had to finish a project in an other new programming language. It was tough but it was the most amazing experience ♥. If you like to get an impression, here is a video about Restart Network.
During the Restart Network program I worked with the following coding languages, tools and frameworks: HTML, CSS, Javascript, PHP, React, Node.js, MySQL, Bootstrap, JQuery, Materialize, Codeigniter and XAMPP. You can find the code of some of those projects on my Github page.
